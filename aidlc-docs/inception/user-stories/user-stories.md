# User Stories

* **As a DevOps Engineer**, I want to paste my raw Slack notes into a form so that I can generate a formal postmortem for my team in seconds.
* **As an Engineering Lead**, I want the output to include a clear "Root Cause Analysis" so that we can prevent the same issue from recurring.
* **As a Job Candidate**, I want to show my ability to handle incidents by generating clean, professional-looking documentation for my portfolio.
* **As a user**, I want to download the result as a PDF so I can easily attach it to Jira tickets or emails.
